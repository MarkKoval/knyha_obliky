# Книга обліку доходів

Веб-додаток для формування Книги обліку доходів з банківської виписки.

## Запуск

```bash
npm install
npm run dev
```

## Можливості

- Імпорт .xls/.xlsx (SheetJS)
- Мапінг колонок (дата, сума, опис)
- Автоматична генерація книги з підсумками
- Таблиця результату на MUI DataGrid
- Експорт у Excel (.xlsx) та PDF
- Збереження останнього документу у localStorage
